# Matrimony Matchmaking System

A console-based Java application that matches 
user profiles using a rule-based engine.

## Tech Stack
- Core Java
- OOP (Encapsulation, Inheritance, Polymorphism, Abstraction)

## Features
- Rule-based matching engine
- Filters by age, income, education, religion, city
- Match scoring system out of 100
- Add, view, search, filter profiles

## Project Structure
- UserProfile.java     - stores person data
- MatchingEngine.java  - rule-based matching logic
- MatchResult.java     - stores match outcome
- MatrimonyApp.java    - main console menu

## How to Run
1. Clone or download this repository
2. Open in Eclipse
3. Run MatrimonyApp.java
